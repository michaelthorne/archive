Style Guide

The style guide is a collection of HTML documents that defines the design and coding rules to determine how the website is structured today, as well as to ensure consistency for future updates.

If you have any questions, submit an issue in our BitBucket repo.

Elements

Elements are the basic building blocks consisting of headings, buttons, icons, etc.

They also include more abstract things like color palettes, font stacks and grids.

Explore

Patterns

Patterns are groups of elements that are combined to function together as a unit.

Building patterns from elements encourages the creation of reusable interface markup.

Explore

Forms

Forms contain interactive controls that are used to submit information to the web server.

These controls can be combined to create patterns such as login and search forms.

Explore

Templates

Templates are built from elements and patterns which have been combined together.

They provide context and are focused on structure as opposed to actual content.

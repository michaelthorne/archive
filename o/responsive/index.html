Style Guide

The style guide is a living set of HTML documents that defines the design and coding rules to determine how the website is structured today, as well as to ensure consistency for future updates.

If you have any questions, submit an issue in our BitBucket repo.

Elements

Elements are the basic building blocks of a website such as headings, buttons, tables and icons.

They include more abstract elements like color palettes and font stacks.

Explore

Patterns

Patterns are groups of elements that are combined to function together as a unit.

Building patterns from elements encourages the creation of reusable interface patterns.

Explore

Forms

Forms contain normal content, markup and special elements called controls.

These elements can be combined to create a common pattern such as search.

Explore

Grid

Grids are used to give structure to the elements and patterns in a layout.

They consist of a series of rows and columns that adjust in size according to the available screen width.

Explore

Templates

Templates are comprised of elements and patterns which have been combined together.

They provide context and are focused on content structure as opposed to actual content.
